# GearBots starterbot typescript

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```

## Publish

Create `apiKey` file in root containing the apiKey of your already existing bot. \
To publish the bot, execute in powershell:
`./publish.ps1` \
You should see `Sucessfully uploaded vX of 'botname'` as a result. \
